# 🔐 Honeyword Generator

A simple cybersecurity tool that generates fake passwords (honeywords) along with a real password to detect unauthorized access attempts.

## 🚀 Features
- Generates multiple realistic fake passwords
- Helps detect password leaks or breaches
- Simple CLI-based tool

## 🛠️ Tech Used
- Python
- Random & String modules
